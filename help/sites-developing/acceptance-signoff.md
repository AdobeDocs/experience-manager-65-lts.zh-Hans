# 验收与签署{#acceptance-and-sign-off}

如前所述，[验收测试](/help/sites-developing/planning.md)是用于验证以下内容的特殊测试类别：

* 项目符合客户的要求。
* 客户认可并接受该项目。

客户和项目经理应定义要求。 由于它们通常与最终发票相关，因此它们都具有特殊的重要性。

因此，要求必须是：

* 尽早定义。
* 必要时更新。
* 合作伙伴和客户都必须正式接受并签署任何更改。
